## Аннотация

We'll talk about DL and tabular data. We'll go from California to $10М seed rounds. I plan to give a broad outlook on the field, trace how it evolved and talk a bit about where it's heading. We'll talk about DL model components specific to tabular data, some speculations and intuitions on why those methods may work, and finish off with a talk about foundation models for tabular data.
